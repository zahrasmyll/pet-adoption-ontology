# Pet Adoption Ontology

## Description
This project aims to develop an ontology for a pet adoption system. 
It models pets, shelters, and adopters, as well as the relationships between them.

## Domain
Pet Adoption System

## Purpose
To organize and represent knowledge about pet adoption processes.

## Scope
The ontology includes pets (such as cats and dogs), shelters, and people. 
It defines relationships such as adoption and shelter assignment.
